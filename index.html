frameworkname

col-md-4 col-sm-10 col-lg-3.... Frameworkname doesn't use that. If you don't feel like rummaging through thousands of lines of css, confusing col-no-thanks classes, or just want something barebones, this is for you. Frameworkname includes core HTML styles and a grid.

variables [sass only]

Life is better with variables, which is why I included a dual-variable configuration with the SASS version of frameworkname.

// Define your color palette
$blue: #00f;
$off-grey: #333;

// Apply your palette here
$button-color: $blue;
$button-primary: $off-grey;

Nothing is more annoying than cmd+f -ing a hex code and replacing all instances, so let the change of 1 variable do it all for you :)

grid

frameworkname uses a 12-column grid with a max width of whatever you want it to be.
The max-width can be changed with one line of CSS, then all columns will resize accordingly.

typography

Frameworkname uses rems for font sizing and ems for spacing when dealing with any non standard font sizes.

The base font asks for HelveticaNeue, falling back to Helvetica, and then Arial. We also have anchors, underlines, italics, and strong.
